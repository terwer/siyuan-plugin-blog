# bat脚本乱码问题解决方案

文件头部加上 `chcp 65001` 即可

常用代码页映射

| 代码页 | 映射的字符集 |
| :----- | :----------- |
| 936    | GB2312       |
| 20127  | US-ASCII     |
| 65001  | UTF-8        |
